# manage-views

> 此后台管理借鉴自 vue-admin-template
> 包含权限管理，动态路由，可能不是尽善尽美，有机会再完善。

#### 使用

```shell
npm install

# 开发环境
npm run dev

# 打包部署
npm run build:prod
```
